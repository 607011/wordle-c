## Build

### Linux, Unix, macOS

Im Terminal:

```
git clone https://github.com/607011/wordle-c.git
cd wordle-c
make
```

### Windows

In der Visual-Studio-Entwickler-Konsole:

```
git clone https://github.com/607011/wordle-c.git
cd wordle-c
nmake exe
```
